chat-ai-guru

Run it locally

```bash
# with pnpm 
pnpm install
pnpm dev

# or with npm
npm install
npm run dev
```

Then open `http://localhost:3000`.

What this is

- Next.js app router project with a simple chat UI and some nice UI primitives.
- No special setup needed beyond Node and your package manager.


Tech stack

- Next.js (App Router)
- React 19 + TypeScript
- Tailwind CSS v4
- ShadCN Components
- TanStack Query
- Zod, React Hook Form
- Lucide icons, Sonner toasts


Features

- Chat interface with Markdown rendering
- Prompt autocomplete and command menu
- Sidebar search and recent conversations
- Light/dark theme toggle
- Keyboard shortcuts
- Local storage for chat history
- Error and loading states
